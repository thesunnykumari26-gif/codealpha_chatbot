Project Title

Basic Chatbot using Python

Description

This project is a simple command-line chatbot built using Python. The chatbot interacts with users by responding to basic messages such as greetings and simple questions. It can greet the user, reply to messages like “hello”, “hi”, and “how are you”, and exit the conversation when the user types “bye”.

This is a beginner-friendly Python project that helps in understanding basic programming concepts like loops, conditional statements, functions, and user input handling.

Features
Interactive command-line chatbot
Responds to basic greetings
Answers simple user questions
Continuous chat until the user types bye
Beginner-friendly and easy to understand
Uses simple Python concepts
Technologies Used
Python 3
Command Line Interface (CLI)
How to Run the Project
Step 1:

Install Python on your computer from
Python Official Website

Step 2:

Create a Python file named:

chatbot.py
Step 3:

Copy and paste the chatbot code into the file.

Step 4:

Open terminal or command prompt and run:

python chatbot.py
